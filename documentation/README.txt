- Unzip the folder IBTVI-Ubuntu.zip
  $ unzip 'IBTVI-ubuntu.zip'

- Enter in the folder and give permissions to the executable file
  $ cd 'IBTVI-ubuntu'
  $ chmod +x IBTVI-ubuntu.x86_64

- Execute the file and enjoy
  $ ./IBTVI-ubuntu.x86_64
